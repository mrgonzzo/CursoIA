Aplicar MCO (M√≠nimos Cuadrados Ordinarios) al dataset wine en Python es un proceso directo, com√∫nmente realizado con la librer√≠a statsmodels.

El objetivo habitual es analizar la relaci√≥n entre el consumo de alcohol de vino (alcohol) y una de las tasas de mortalidad, 
como las muertes por enfermedad card√≠aca (heart) o hep√°tica (liver).

Aqu√≠ te explico c√≥mo aplicar MCO para estimar el impacto de alcohol sobre heart.

1. Importar Librer√≠as y Datos üç∑

Necesitas pandas para manejar los datos, wooldridge para cargarlos y statsmodels.formula.api para la regresi√≥n.
Python

import pandas as pd
import statsmodels.formula.api as smf
import wooldridge

# Cargar el dataset 'wine'
df = wooldridge.data('wine')

2. Aplicar MCO (Regresi√≥n Lineal Simple) üìà

Utilizaremos la sintaxis de f√≥rmula de statsmodels, donde la ecuaci√≥n se define como VariableDependiente ~ VariableIndependiente.

Modelo a Estimar: 
heart=Œ≤0‚Äã+Œ≤1‚Äã‚ãÖalcohol+u
Python

# Definir la f√≥rmula del modelo
formula = 'heart ~ alcohol'

# Aplicar MCO: OLS (Ordinary Least Squares)
modelo_mco = smf.ols(formula, data=df)

# Entrenar el modelo
resultados = modelo_mco.fit()

3. Mostrar e Interpretar los Resultados üìã

El m√©todo .summary() proporciona la tabla de resultados econom√©tricos completa.
Python

# Mostrar el resumen de los resultados
print(resultados.summary())

Interpretaci√≥n Clave (Basada en la pr√°ctica com√∫n):

    Coeficiente de alcohol (Œ≤1‚Äã): El valor te indicar√° si, por cada litro adicional de alcohol de vino consumido per c√°pita, el n√∫mero de muertes por enfermedad card√≠aca por 100,000 personas aumenta (si es positivo) o disminuye (si es negativo).

    P-valor (P>|t|): Si este valor es menor que 0.05 (o el nivel de significancia elegido), el efecto se considera estad√≠sticamente significativo.

    R-cuadrado: Mide el porcentaje de la variaci√≥n total en la tasa de muertes card√≠acas que es explicada por el consumo de alcohol de vino.

    üí° Consejo Adicional: Para analizar el otro lado de la "paradoja del vino" (efecto negativo en el h√≠gado), simplemente cambia la variable dependiente a liver: formula = 'liver ~ alcohol'.
